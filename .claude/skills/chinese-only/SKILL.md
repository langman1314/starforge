# 全中文模式（强制生效）

## 核心规则

用户**看不懂任何英文**。本AI**所有输出必须使用简体中文**。

---

## 具体规则

### 1. 思考过程必须全中文
所有推理、分析、决策、计划，全部用中文书写。禁止出现任何英文思考。

### 2. 回复内容必须全中文
对用户的每一句话，全部用中文。一个英文字母都不能出现。

### 3. 工具调用参数必须全中文
- `description` 字段必须写中文
- 其他用户可见参数字段也必须写中文

### 4. 写入文件的内容必须全中文
.md、.txt等文件全部写中文。代码注释写中文。

### 5. 允许保留英文的特例（仅限以下情况）
- 代码语法关键词：`if`、`for`、`function`、`import`、`return`等
- 文件扩展名：`.md`、`.txt`、`.json`、`.py`等
- 工具/API标识符：`Bash`、`Read`、`Write`、`mcp__claw__notify`等工具名
- 国际通用缩写：`AI`、`CPU`、`URL`、`API`、`ID`等

### 6. 禁止出现的英文内容
- ❌ 思考过程中的英文
- ❌ 回复中的英文单词/短语/句子
- ❌ 工具调用描述中的英文
- ❌ 文件名中多余的英文

---

## 生效范围

| 内容类型 | 是否强制中文 |
|----------|------------|
| 思考过程 | ✅ 强制 |
| 文本回复 | ✅ 强制 |
| 工具参数 | ✅ 强制 |
| 文件内容 | ✅ 强制 |
| 代码语法 | ❌ 保持原样 |
| 工具名称 | ❌ 保持原样 |
